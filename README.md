# Murray et al. (2026) LysH1

This repository contains scripts to plot phylogenetic trees of cross-class activity for LysH1.

## Contents

**custom_taxonomy.txt** - Representative genomes used in this study, including those tested for cross-class activity versus LysH1.

**gtdb_tree_nr_genomes.smk** - Snakemake file to generate the GTDB tree.
